> 基于《色彩的性格》心理学研究
> 版本：v1.0 | 更新：2026-04-18

## 游戏流程

1. **首页** → 点击「开始探索你的色彩人生」
2. **8道测验题** → 回答偏好
3. **人格揭示** → 显示你的主色彩人格
4. **5个人生场景** → 每个场景做选择
5. **最终评价** → 满足度 + 人生构成图 + 抉择回顾

## 8道测验题（Quiz）

### Q1. 走进一间房间，墙壁颜色你感觉最舒服？
- 纯白极简
- 暖黄灯光咖啡馆
- 绿植环绕自然空间
- 深蓝海景
- 紫色霓虹未来感

### Q2. 和朋友有分歧，你会怎么做？
- 直接约出来当面聊清楚
- 先冷静分析再沟通
- 算了没必要争主动让步
- 先表达理解再委婉说

### Q3. 计划旅行，最想去？
- 热带海岛潜水阳光
- 欧洲古城历史艺术
- 深山徒步帐篷星空
- 日本小镇安静仪式感
- 纽约东京大都市繁华

### Q4. 职场最被认可的是？
- 有魄力关键时刻冲得上
- 分析稳健让人放心
- 点子多总有新想法
- 善于协调化解矛盾

### Q5. 哪个词最能概括你的理想生活？
- 自由不束缚按节奏生活
- 深度有意义的连接体验
- 温暖有爱人有归属感
- 做成大事证明自己价值
- 每天被美好事物包围

### Q6. 面对巨大压力，你更可能？
- 睡不着想各种最坏可能
- 列计划一步步解决
- 找朋友倾诉寻求支持
- 运动出汗用身体带走压力

### Q7. 你的另一半最吸引你的特质？
- 魄力带我看更大世界
- 温和体贴懂我的需要
- 有思想深度能认真对话
- 幽默有趣永远不无聊

### Q8. 做团队领导，最看重什么？
- 执行力说干就干不拖
- 专业度在领域内足够深入
- 协同力配合团队目标
- 创意带来不一样视角

## 8种色彩人格

### ❤️ 红 — 生命的燃烧者
**颜色值：** #e74c3c
**描述：** 你热情直接，爱恨分明，活在当下。你的真实让人无法忽视。
**特质：** 热情、冒险、直接

### 🩵 蓝 — 信任的守护者
**颜色值：** #3498db
**描述：** 你深思熟虑，重视深度，善于倾听。你是朋友遇到困难时第一个想到的人。
**特质：** 理性、忠诚、深沉

### 💛 黄 — 阳光的传播者
**颜色值：** #f1c40f
**描述：** 你自带光芒，乐观好奇，思维跳跃。和你在一起永远不会无聊。
**特质：** 乐观、好奇、跳跃

### 💚 绿 — 和平的践行者
**颜色值：** #27ae60
**描述：** 你追求平衡，不争不抢，是身边人的定心丸。你用包容化解冲突。
**特质：** 平和、包容、稳定

### 🖤 黑 — 优雅的掌控者
**颜色值：** #2c2c2c
**描述：** 你神秘有品位，不轻易让人看透。你的气场本身就是一种力量。
**特质：** 神秘、掌控、品位

### 💜 紫 — 创意的梦想家
**颜色值：** #9b59b6
**描述：** 你不属于任何常规格子，思维方式总是出人意料。你是最难被定义的存在。
**特质：** 创意、独特、梦想

### 🧡 橙 — 活力的冒险家
**颜色值：** #e67e22
**描述：** 你浑身是劲，敢想敢干。你的自信能感染周围每一个人。
**特质：** 活力、自信、行动

### 🩷 粉 — 温柔的力量者
**颜色值：** #fd79a8
**描述：** 你用柔软改变世界，不需要强势就能让人跟随。这是一种难得的能力。
**特质：** 温柔、敏感、关怀

## 5个人生场景（Scenes）

### 场景1：毕业择业
**描述：** 两个offer摆在面前：大公司稳定岗 VS 创业公司核心职位。

| 选项 | 副标题 | 匹配色彩 |
|------|--------|---------|
| 创业公司核心职位 | 风险高，成长空间大 | ❤️红 |
| 大公司稳定岗位 | 稳妥优先，积累经验 | 🩵蓝 |
| 两个都去试试再说 | 先体验，不纸上谈兵 | 🧡橙 |
| 跟父母商量再决定 | 不想一个人做重大决定 | 🩷粉 |
| 想想哪个更酷 | 想做不一样的 | 💜紫 |
| 哪个对简历有用？ | 现实主义，实用优先 | 🖤黑 |
| 先观望收集更多信息 | 不急着选 | 💛黄 |
| 先休息一段时间再说 | 不急着决定 | 💚绿 |

## 最终评价系统

### 满足度计算
- 每道测验题：匹配主色彩 +20分，其他色彩 +10分
- 满分：100分（5场景 × 20分）
- 满足度百分比 = 积分 / 100

### 评价等级

| 满足度 | 评价 | 图标 |
|--------|------|------|
| ≥80% | 人生赢家 | 🏆 |
| 60-79% | 精彩人生 | 🌟 |
| 40-59% | 平凡真实 | 💫 |
| 20-39% | 迷失探索 | 🌙 |
| <20% | 寻找自我 | 💀 |
